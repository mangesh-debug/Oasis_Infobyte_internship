# 🚀 Java Development Internship Projects - Oasis Infobyte

This repository contains all the tasks completed during my **1-month Java Development Internship at Oasis Infobyte**. These projects helped me strengthen my understanding of **core Java, object-oriented programming, and backend logic development**.

---

## 📌 Projects Included

### 1️⃣ Online Reservation System 🎟️
A console-based application that allows users to:
- Book tickets
- Cancel reservations
- View reservation details

**Concepts Used:**
- Classes & Objects
- Collections (HashMap)
- User Input Handling

---

### 2️⃣ Number Guessing Game 🎯
An interactive game where:
- System generates a random number
- User guesses the number
- Hints like *too high* or *too low* are provided

**Concepts Used:**
- Random Class
- Loops & Conditions
- Input Validation

---

### 3️⃣ Online Examination System 📝
A simple quiz system with:
- User login functionality
- Multiple-choice questions
- Score calculation

**Concepts Used:**
- ArrayList
- Conditional Logic
- Basic Authentication

---

### 4️⃣ Digital Library Management System 📚
A system to manage library books:
- Add books
- Search books
- Issue and return books

**Concepts Used:**
- Object-Oriented Programming
- Data Structures (HashMap)
- String Handling

---

### 5️⃣ ATM Interface 🏧
A basic ATM simulation that supports:
- Login using PIN
- Balance checking
- Deposit and withdrawal

**Concepts Used:**
- Classes & Objects
- Conditional Statements
- Transaction Handling
